Comment la pandémie de la COVID-19 a-t-elle impactée les collaborations scientifiques ?

Presentation du projet

La pandémie a complètement redéfini les collaborations scientifiques à l'échelle mondiale. A travers ce site web, plongez dans l’analyse de la base de données DBLP (Digital Bibliography and Library Project), une base de données bibliographiques dans le domaine de l’informatique, qui recense plus de 7 millions de publications scientifiques et plus de trois millions d’auteurs.
